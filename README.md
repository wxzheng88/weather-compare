# 天气预报对比工具

一个简单实用的天气预报对比网页应用，支持对比黑龙江省安达市和甘南县未来5天的天气情况。

## 功能特性

- **双城对比**：左右布局，同时显示安达市和甘南县的天气
- **未来预报**：展示未来5天的天气预报
- **完整数据**：
  - 最高/最低温度
  - 体感温度
  - 降水概率
  - 降水量
  - 风速
  - 湿度
  - 紫外线指数
  - 日出/日落时间
- **交互设计**：点击天气卡片可展开查看详情
- **响应式布局**：支持桌面端、平板和手机
- **自动刷新**：手动刷新获取最新天气数据

## 技术栈

- **HTML5** - 语义化页面结构
- **CSS3** - 现代渐变风格，动画效果
- **JavaScript (ES6+)** - 核心逻辑
- **FontAwesome** - 图标库
- **Open-Meteo API** - 天气数据源（免费，无需API Key）

## 使用方法

### 在线访问

访问 GitHub Pages 部署的在线版本：
```
https://你的用户名.github.io/weather-compare/
```

### 本地运行

1. 克隆或下载项目文件
2. 直接用浏览器打开 `index.html` 文件
3. 无需服务器，直接在浏览器中运行

### 刷新数据

点击页面底部的刷新按钮（🔄）可获取最新的天气数据。

## 数据来源

天气数据由 [Open-Meteo](https://open-meteo.com/) 提供。

- ✅ 完全免费
- ✅ 无需注册/API Key
- ✅ 支持全球天气数据
- ✅ 高精度预报
